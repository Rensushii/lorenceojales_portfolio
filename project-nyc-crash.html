NYC Crash Analytics

Full‑stack interactive dashboard for motor‑vehicle collision data

View Live Site

Overview

NYC Crash Analytics is a full‑stack web dashboard that transforms raw NYC Open Data into an interactive visual exploration tool. Built with PHP, MySQL, and vanilla JavaScript on the front end, it lets users filter hundreds of thousands of motor‑vehicle collision records by borough, date, vehicle type, contributing factor, and severity.

Key Features

A responsive dark‑theme interface presents a live Leaflet map with marker clustering, heatmaps, a time‑slider, and on‑map drawing tools for spatial filtering. Side‑by‑side, a paginated data table shows detailed records, while integrated Chart.js visualizations reveal trends, borough breakdowns, and injury summaries.

Under the Hood

A custom data importer pulls fresh crash data from the NYC Open Data API, supports multi‑year imports, and allows users to delete or replace existing data via a management modal. The project also includes secure user authentication, CSV/JSON/PDF exports, and a built‑in guided tour for onboarding.
